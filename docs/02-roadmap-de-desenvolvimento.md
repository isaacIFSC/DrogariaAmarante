# Roadmap de desenvolvimento

## Fase 1 - Base funcional do e-commerce
Objetivo: deixar o projeto minimamente navegavel para clientes.

- Implementar listagem completa de produtos.
- Implementar busca e filtros.
- Criar pagina de detalhe do produto.
- Criar carrinho com adicionar, remover e alterar quantidade.
- Exibir totais do carrinho.

## Fase 2 - Conta do cliente
Objetivo: permitir identificacao e historico.

- Cadastro de usuario.
- Login e logout.
- Hash de senha seguro.
- Perfil do cliente.
- Historico de pedidos.

## Fase 3 - Checkout e pedidos
Objetivo: transformar carrinho em compra real.

- Criar entidade Pedido.
- Criar itens do pedido.
- Definir status de compra.
- Criar fluxo de checkout.
- Reservar e baixar estoque.

## Fase 4 - Pagamentos e integracoes
Objetivo: aproximar o projeto de uma operacao real.

- Integrar Pix.
- Integrar cartao.
- Integrar debito.
- Registrar confirmacao de pagamento.
- Preparar suporte a boleto, se necessario.

## Fase 5 - Administracao
Objetivo: permitir gestao interna.

- CRUD de produtos.
- CRUD de categorias e marcas.
- Gestao de estoque.
- Gestao de pedidos.
- Gestao de cupons e promocoes.

## Fase 6 - Inteligencia operacional
Objetivo: apoiar decisao e crescimento.

- Relatorios de vendas.
- Relatorios de estoque.
- Produtos mais vendidos.
- Clientes ativos.
- Painel resumido.

## Fase 7 - Qualidade e producao
Objetivo: preparar para evolucao sustentavel.

- Testes automatizados.
- Tratamento centralizado de erros.
- Logs e auditoria.
- CI/CD.
- Documentacao tecnica e de uso.

## Ordem recomendada de execucao imediata
1. Catalogo com busca e filtros.
2. Carrinho com escrita.
3. Cadastro e login.
4. Pedido e checkout.
5. Admin e estoque.
